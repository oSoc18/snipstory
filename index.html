The project

Snipstory is a project that started in Open summer of code 2017 — during Open summer of code 2018 the project was picked up again and expanded.

A team of five students worked on it for four more weeks, to add new functionalities to the web application.

Snipstory allows musea and other contentpartners to add stories with a focus on normal people in extraordinary times.

Teachers can manage those stories and use them in class, and pupils can get creative and tinker with history.
